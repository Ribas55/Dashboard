# SKU Intelligence Platform — Sales Analytics Dashboard

An interactive **Streamlit dashboard** for sales analytics, demand classification, and forecasting at **SKU level**, designed to support **inventory planning, production decisions, and demand forecasting improvement** in multi-SKU environments.

This project demonstrates practical application of **time-series analysis, demand segmentation, and forecasting model evaluation** using real-world analytics workflows.

---

## Features

* Interactive **sales time-series dashboards**
* **ABC/XYZ demand classification**
* **Intermittency analysis** (ADI & CV²)
* Multiple **forecasting models** with performance comparison
* **Weighted ensemble forecasting**
* Automated **SKU-level insights and recommendations**

---

## Forecasting Methods

* Simple Moving Average (SMA)
* Single Exponential Smoothing (SES)
* ARIMA
* Linear Regression
* XGBoost
* TSB (for intermittent demand)

Performance evaluated using **MAE, RMSE, and MAPE**.

---

## Tech Stack

Python • Streamlit • pandas • numpy • statsmodels • scikit-learn • XGBoost • matplotlib • plotly

---

## Context

Built as part of a **sales analytics and demand forecasting project** focused on improving forecasting accuracy across a **multi-SKU product portfolio** using **classification-driven modeling**.
